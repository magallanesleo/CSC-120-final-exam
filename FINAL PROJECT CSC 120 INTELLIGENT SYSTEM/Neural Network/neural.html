About Dataset

Content

Zalando article photos make up the Fashion-MNIST dataset, which has a training set of 60,000 samples and a test set of 10,000 examples. Each illustration is a 28x28 grayscale graphic paired with a label drawn from one of ten classes. When testing machine learning methods, Zalando wants Fashion-MNIST to be a simple drop-in substitute for the original MNIST dataset. The image size and split structure between training and testing are the same.
